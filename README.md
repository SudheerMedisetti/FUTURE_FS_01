# Future Interns - Portfolio Website Task

## 👤 Developer: Sudheer
## 🎯 Goal: Personal Portfolio Website (Task 1)

This is a modern, responsive, and professional portfolio website built using ONLY HTML, CSS, and Vanilla JavaScript. It features a deep-space dark theme with glassmorphism aesthetics, smooth scrolling, and scroll-triggered animations.

---

### 🚀 Features

- **Hero Section**: Animated gradients and a strong CTA.
- **Glassmorphism UI**: Modern aesthetic using backdrop-filters and semi-transparent surfaces.
- **Responsive Design**: Fully optimized for Mobile, Tablet, and Desktop.
- **Smooth Navigation**: Sticky navbar with scroll-spy and internal link smoothing.
- **Form Validation**: Real-time feedback for the contact form.
- **Scroll Animations**: Elements reveal themselves as you scroll using the Intersection Observer API.

### 🔧 Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/FUTURE_FS_01.git
   ```
2. **Open the project**:
   Simply open `index.html` in any modern web browser (Chrome, Edge, Firefox).
3. **Customize Content**:
   - Edit `index.html` to update your personal details and project links.
   - Replace `assets/profile.jpg` with your actual photo.
   - Replace `assets/resume.pdf` with your professional resume.

---

### 🌐 Deployment (GitHub Pages)

1. Create a new repository on GitHub named `FUTURE_FS_01`.
2. Push your code to the repository:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Portfolio Website"
   git remote add origin https://github.com/your-username/FUTURE_FS_01.git
   git branch -M main
   git push -u origin main
   ```
3. Go to the **Settings** tab of your repository.
4. Click on **Pages** in the left sidebar.
5. Under **Branch**, select `main` and `/root`, then click **Save**.
6. Your site will be live at `https://your-username.github.io/FUTURE_FS_01/` after a few minutes!
